# frontend/static/css

프론트엔드 스타일시트 폴더입니다.

---

## 디렉토리 구조

```
css/
├── auth/
│   └── login.css          # 로그인 페이지 전용 스타일
└── common/
    ├── nav.css             # 전체 페이지 공통 네비게이션 폰트
    ├── dino-loader.css     # 공룡 로딩 애니메이션 컴포넌트
```

---

## 폴더별 역할

| 폴더      | 용도                                               |
| --------- | -------------------------------------------------- |
| `auth/`   | 특정 페이지 전용 스타일. 현재 로그인 페이지만 존재 |
| `common/` | 여러 페이지에서 공유하는 스타일                    |

---

## 스타일 적용 방식

이 프로젝트는 페이지별 스타일을 주로 **Tailwind CSS 인라인 클래스**로 처리합니다.
이 폴더의 CSS 파일들은 Tailwind로 처리하기 어려운 경우에만 사용합니다.

- `nav.css` — 폰트 CSS 변수 선언 및 헤더·사이드바 폰트 통일 (Tailwind로 처리하기 어려운 전역 변수)
- `login.css` — 로그인 페이지 카드 레이아웃 (독립적인 페이지 스타일)
- `dino-loader.css` — 프레임 스프라이트 애니메이션 (복잡한 `@keyframes`, Tailwind로 표현 불가)

자세한 내용은 각 하위 폴더의 README 참고.
