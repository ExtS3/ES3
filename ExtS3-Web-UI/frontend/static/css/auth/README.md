# frontend/static/css/auth

로그인 페이지 전용 스타일 폴더입니다.

---

## 파일 구성

### login.css

`frontend/templates/auth/login.html` 한 곳에서만 참조합니다.

**담당 스타일**

| 클래스 / 셀렉터             | 설명                                             |
| --------------------------- | ------------------------------------------------ |
| `body`                      | 배경색(`#f7f9fc`), 폰트, 여백 초기화             |
| `.login-shell`              | 화면 중앙 정렬 컨테이너 (flex, min-height 100vh) |
| `.login-panel`              | 흰색 카드 패널 (최대 너비 420px, 그림자, 라운드) |
| `.brand-mark`               | 좌측 상단 브랜드 아이콘 박스 (보라색 정사각형)   |
| `.login-form`               | 폼 그리드 레이아웃                               |
| `.field`                    | 라벨 + 인풋 묶음                                 |
| `input`                     | 포커스 시 보라색 테두리·글로우 효과              |
| `button`                    | 보라색 제출 버튼 (호버·액티브 트랜지션)          |
| `.signup-link`              | 하단 회원가입 링크                               |
| `@media (max-width: 480px)` | 모바일 패딩 축소                                 |

> `nav.css`도 `login.html`에 함께 포함돼 있지만, 로그인 페이지에는 네비게이션 바가 없으므로 실질적으로 적용되는 스타일이 없습니다. 추후 정리 대상입니다.
