Chrome 정책 카탈로그

5종 표준 Chrome 관리 정책을 입력 → 미리보기 → .bat 다운로드. 사용자 PC에서 관리자 권한으로 실행하면 레지스트리에 정책이 등록됩니다.

category

정책 타입

code

입력 JSON

정책 타입을 바꾸면 예시 JSON으로 자동 채워집니다. 자유롭게 수정하세요.

terminal

생성된 batch 미리보기

미리보기 버튼을 누르면 결과가 여기에 표시됩니다.
info

레지스트리 경로

모든 정책이 등록되는 루트:

HKLM\Software\Policies\Google\Chrome

warning

실행 주의

  • 다운로드한 .bat는 Windows에서 관리자 권한이 필요합니다.
  • 실행 시 사용자 PC의 Chrome 정책이 즉시 변경됩니다 — 테스트 머신에서만 실행.
  • 적용 후 Chrome 재시작 필요.
